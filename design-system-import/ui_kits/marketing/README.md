# WORKO Marketing — UI Kit

The WORKO marketing landing page. Open `index.html`.

Sections (`Marketing.jsx`): sticky nav, hero (tagline + dual CTA), a product-preview card showing the Focus Home hero, a 2×2 feature grid, an indigo CTA band, and footer.

Copy follows WORKO's voice — sentence case, verb-first CTAs, taglines in the one allowed Title Case spot. Components and tokens come from the design system bundle (`window.WORKODesignSystem_3ef60c`); icons are Lucide via the shared `../app/icons.jsx`.
