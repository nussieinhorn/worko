A clean single-line text input with an indigo focus ring — the default field for forms, search, and task entry.

```jsx
<Input label="Task name" placeholder="What needs doing?" hint="Press enter to add" />
<Input label="Email" type="email" error="That email looks off" />
```

Optional `label`, `hint`, `error` (turns the border red), and `leadingIcon`. Forwards native input props.
